Talairach Daemon Interface - User Guide

The Research Imaging Center, University of Texas Health Science Center at San Antonio provides programs which take in Talairach co-ordinates and then interrogate their Talairach labelling database over the internet (their database is accessible directly via the The Talairach Daemon). We have integrated these into MEDx.

Warning:

  • There appears to be an inconsistency between some supposedly "Talairached" brains (e.g., MNI 305) and the Talairach daemon labels
  • Please note the objection that some people make to the whole philosophy behind this kind of procedure, i.e., misgivings about relying on any automated method of anatomic labelling.

To use, bring up an image in Talairach space, and then select MEDx->FMRIB->General->Talairach Daemon. MEDx will be put into 3D Marker mode. Use the left mouse button to place a marker anywhere in the image. The marker can also be moved using the left button once placed. Then press the Click ... press here button, and the position will be processed and reported. This can be repeated many times with new markers. All markers and all reports can be cleared using the Clear button. Note that two types of data are returned, Talairach labels (from the atlas) and Structure Probability Map probabilities (from many segmented brains).

The programs used are available from http://ric.uthscsa.edu/projects/registration.
